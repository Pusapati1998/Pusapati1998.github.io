Overview

This project explores job listings to uncover demand by title and city, trending skills, and salary distributions. It includes cleaning, EDA, text analysis for skills, and rich visualizations. (See README for objectives, dataset schema, and tooling details.)

  • Objectives: demand by role & location, skills trends, salary ranges.
  • Tech: Python, Pandas/NumPy, Matplotlib/Seaborn/Plotly, NLTK, Jupyter.

Method & Highlights

  • Data cleaning: nulls/duplicates handled, salary & experience parsed, multi-valued skills tokenized.
  • EDA: job counts by city/title/company, salary distributions by role & level.
  • Text analysis: frequent technical keywords / buzzwords across roles.
  • Visuals: bar charts, histograms, correlation views, word clouds, and maps.

Key Insights (sample)

  • Top skills include Python, SQL, Excel, AWS; higher salaries trend with cloud/AI roles.
  • Hiring hotspots include SF, NYC, and Seattle; remote roles increasing.
