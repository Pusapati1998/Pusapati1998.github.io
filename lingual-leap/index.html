Lingual Leap

ISAD Project • System Design • Prototyping • Cost-Benefit Analysis

Overview

Lingual Leap is a proposed language-learning app designed as part of the ISAD course. The project covers system diagrams, cost-benefit analysis, and prototype UI designs created with Lucidchart, Excel/Sheets for analysis, and Figma/Adobe XD for UI prototyping.

  • Course: ISAD (Information Systems Analysis & Design)
  • Tools: Lucidchart, Excel (CBA), Figma/Adobe XD (Prototypes)
  • Team: Padma Rishitha Pusapati & Group Members
