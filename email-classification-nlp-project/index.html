Email Classification System

Spam Detection • Priority Classification — Python / NLP / ML

Overview

This project applies Natural Language Processing (NLP) techniques to classify emails as spam or ham, and further assign priority levels (High, Medium, Low). It includes dataset collection from Gmail API, preprocessing, model building, and performance evaluation.

  • Objectives: spam detection and priority classification.
  • Tech: Python, Pandas, NLTK, Scikit-learn, TF-IDF, Logistic Regression, Random Forest, Jupyter.

Method & Highlights

  • Dataset from Gmail API: subject, sender, date, body.
  • Preprocessing: stopword removal, tokenization, lemmatization, lowercasing.
  • Feature Engineering: Bag of Words, TF-IDF, Word2Vec.
  • Models: Logistic Regression, Random Forest, Naive Bayes, XGBoost, SVM.
  • Evaluation: accuracy, precision, recall, F1-score, confusion matrices.

Key Insights (sample)

  • Logistic Regression with TF-IDF gave best results for spam detection.
  • Random Forest was less balanced for priority classification compared to Logistic Regression.
  • Spam detection achieved over 95% accuracy; priority classification struggled most with “High” class.
